The "Bionicisms" Framework

How Moltis overcomes the strict Android sandbox to provide a flawless VPS environment.

Phantom Process Killer

Android 12+ randomly murders child processes like AI engines or Git actions. Moltis requires this ADB fix to preserve your background tasks permanently.

adb shell "settings put global phantom_process_handling false"
üöè

The Hardcoded Path Trap

There is no `/bin/bash` on Native Android Bionic. Moltis intercepts and injects `termux-exec` (`LD_PRELOAD`) so standard scripts magically run correctly.

export LD_PRELOAD=$PREFIX/lib/libtermux-exec.so

No Systemd (Runit Injection)

Phones don't have `systemd`. We orchestrate everything using `termux-services` (a RunIt implementation) so crashing bots auto-restart automatically.

sv up moltis && svlogd -tt $PREFIX/var/log/moltis
